# 설치 및 실행 방법

프론트엔드

```bash
git clone https://github.com/jeehwan98/FRONT-LOGIN
npm install
npm start
```

백엔드

```bash
git clone https://github.com/jeehwan98/BACK-LOGIN
./mvnw spring-boot:run
```
